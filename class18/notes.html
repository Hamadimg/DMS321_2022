Client-Side Javascript

There are two important tools related to coding in the browser:

  • DOM (document object model)
  • Events





    Document Object Model

    An element can be accessed by first finding it via its ID: 

        let elt = document.getElementById('foo');

    or: 

        function $(name) { return document.getElementById(name); }

    let elt = $('foo');


    Then, the element's attributes can be modified 

        elt.textContent = 'hello, world!';
    imgelt.src = 'http://ubdms321.xyz/sasha.jpg';
    hdrelt.style.color = 'red';

    Events

    Events happen on elements while the page is being viewed
    They are available via the "on" attributes of the elements

    Some common events:

    • click
    • mouseover
    • mouseout
    • keyup
    • keydown
    • focus
    • blur

    Example: 

    <span onmouseover="this.style.background='yellow';" onmouseout="this.style.background='white'">test</span>

    <span onclick="$('para3').innerHTML='new text';">test2</span>

    Timing

    Timing events are used to make something happen repeatedly, or after a delay

    • setInterval(func, milliseconds)
    • setTimeout(func, milliseconds)

    (These can be stopped with "clearInterval()" or "clearTimeout()")
