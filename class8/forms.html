HTML Forms

input to website: person viewing page can submit data

important parts:

  • action
  • method
  • inputs: name -> value







action

URL to which form data is sent


method

GET
includes data in the URL. example:
http://example.com/myscript?firstname=John&lastname=Doe
POST
data is sent in HTTP header
allows for more data






Form inputs

all inputs should have a unique "name" attribute
name is used on the server to access the different data

some major input types:

submit button
sends the form to the server
<input type="submit" name="submit">


text input
<input type="text" name="firstName">


checkbox
<input type="checkbox" name="acceptedeula" value="1">


textarea
<textarea name="blockoftext" rows="10" cols="40"></textarea>


menu
<select name="menuchoice">
<option value="1">one</option>
<option value="2">two</option>
<option value="not3">four</option>
</select>


hidden input
<input type="hidden" name="secret" value="fnord">
